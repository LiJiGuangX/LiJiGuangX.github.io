# 高考志愿咨询师系统 各端核心页面文字原型（UI设计专用）

核心说明：所有页面遵循“极简、低学习成本、贴合方案”原则，明确标注「位置」「字段/元素」「功能说明」，UI可直接参照布局，无需额外拆解；重点突出关键词、段位title、协议入口等新增功能。

# 一、咨询师端（导师端）核心页面

## 页面1：咨询师注册认证页（核心注册页）

布局：垂直居中，从上到下依次排列，底部固定提交按钮

- 【顶部】标题：咨询师注册认证（居中，加粗）

- 【表单区】（垂直排列，每一项占满页面宽度，间距均匀）
        

    - 字段1：姓名（输入框，提示文字：请输入真实姓名，必填）

    - 字段2：身份证号（输入框，提示文字：请输入18位身份证号，必填）

    - 字段3：院校（输入框/下拉选择，提示文字：请输入/选择毕业/在读院校，必填）

    - 字段4：专业（输入框/下拉选择，提示文字：请输入/选择所学专业，必填）

    - 字段5：学信网验证（按钮+上传框，提示文字：请上传学信网验证截图/输入验证码，必填；按钮文字：上传验证材料）

- 【底部固定】按钮：提交认证（居中，加粗；点击后跳转至“个人宣传设置页”）

- 【底部小字】提示：认证信息将严格审核，审核通过后方可开通咨询权限（居中，灰色字体）

## 页面2：个人宣传设置页（核心优化页）

布局：左侧表单区，右侧预览区（PC端）；移动端垂直排列，预览区在表单区下方

- 【顶部】标题：个人宣传设置（居中，加粗）

- 【左侧/上方-表单区】（垂直排列，间距均匀）
        

    - 1. 自我介绍（文本框，提示文字：请简要介绍自己的咨询优势、经历，必填，限500字内）

    - 2. 咨询范围（文本框，提示文字：请填写可咨询的内容的范围，必填；右侧新增【协议确认】按钮，按钮旁小字：点击查看平台与导师协议）
                

        - 协议弹窗（点击按钮弹出）：标题“平台与咨询师协议”，内容为方案中约定的禁止项+处罚规则，底部勾选框“我已阅读并同意以上协议”（必勾选方可提交）

    - 3. 关键词设置（核心新增）
                

        - 字段1：一级关键词（多选框，提示文字：请选择平台预设关键词，可多选）；选项为方案中4大类一级关键词（志愿填报类、学业发展类等，每个类别下列具体关键词）

        - 字段2：二级关键词（输入框，提示文字：请填写个性化关键词，多个用逗号分隔，需平台审核，非必填）

        - 提示小字：关键词将用于用户检索，请如实填写，违规关键词将审核不通过

    - 4. 宣传材料上传
                

        - 字段1：图片上传（按钮+预览区，提示文字：可上传成绩、奖项、科研等相关图片，最多5张；按钮文字：上传图片）

        - 字段2：视频上传（按钮+预览区，提示文字：可上传自我介绍、资质相关视频，最多1个，时长不超过5分钟；按钮文字：上传视频）

    - 5. 定价设置（输入框，提示文字：请输入咨询单价（元/30分钟），必填；右侧提示：平台建议价格：50-200元/30分钟）

- 【右侧/下方-预览区】（模拟用户端看到的导师主页，实时同步表单内容）
        

    - 显示：头像（默认头像，可后续修改）、姓名、身份标签（在校生/毕业3年内）、院校+专业、段位title（默认“青铜咨询师”）、咨询单价、关键词标签、自我介绍、宣传材料缩略图

- 【底部固定】按钮：保存并提交审核（居中，加粗；点击后提示“提交成功，等待平台审核”）

## 页面3：可预约时间设置页

布局：顶部标题+说明，中间日历选择区，底部保存按钮

- 【顶部】标题：可预约时间设置（居中，加粗）

- 【顶部小字】提示：请设置可提供咨询的时间段，每次咨询时长为30分钟，用户将按时间段预约（居中，灰色字体）

- 【中间】日历选择区（核心）
        

    - 显示：月历视图（可切换月份），点击日期可选择“可预约日期”（选中日期标亮）

    - 选中日期后，下方显示“可预约时间段”（下拉选择/手动输入，默认选项：09:00-09:30、09:30-10:00...21:00-21:30，可勾选多个时间段）

    - 按钮：批量设置（可设置每周固定日期+时间段，减少重复操作）

- 【底部固定】按钮：保存设置（居中，加粗；点击后提示“设置成功”）

## 页面4：我的等级与成长页（核心新增）

布局：顶部段位展示，中间升级条件，底部成长说明

- 【顶部】当前段位展示（居中，突出显示）
        

    - 大字体：当前段位（如“黄金咨询师”）（加粗，颜色突出）

    - 小字：后台等级（如“3级”）+ 距离下一级还差XX成长值（灰色字体）

- 【中间】升级条件（垂直排列，仅导师可见，用户不可见）
        

    - 标题：升级至下一级（如“钻石咨询师”）条件（加粗）

    - 字段1：咨询完成率≥95%（当前：98%）

    - 字段2：平台裁决好评率≥90%（当前：88%）

    - 字段3：无违规记录（当前：无）

    - 字段4：累计咨询时长≥100小时（当前：85小时）

- 【底部】成长说明（小字，灰色）：等级提升可解锁专场直播申请权限，获得更多曝光和佣金加成

- 【右侧固定】按钮：申请专场直播（仅达到对应等级可点击，未达标时置灰，提示“等级不足，无法申请”）

## 页面5：预约管理页

布局：顶部筛选栏，中间预约列表，每条列表右侧操作按钮

- 【顶部】筛选栏（水平排列）
        

    - 筛选选项：全部预约、待确认、已确认、已完成、已取消

    - 搜索框：输入用户姓名/预约日期检索（placeholder：搜索用户/日期）

- 【中间】预约列表（每条列表水平排列，包含以下字段）
        

    - 字段1：用户姓名（隐藏部分字符，如“李**”）

    - 字段2：预约日期+时间段（如“2026-06-10 19:00-19:30”）

    - 字段3：预设问题（简短展示，如“咨询计算机专业保研流程”）

    - 字段4：预约状态（待确认/已确认等，不同状态不同颜色标注）

    - 【每条右侧】操作按钮（根据状态显示）
                

        - 待确认：接受预约 / 取消预约（点击取消需填写原因）

        - 已确认：发起会议（预约时间前10分钟可点击） / 取消预约（需承担违约处罚）

        - 已完成：查看会议笔记

# 二、用户端（家长/学生端）核心页面

## 页面1：首页（核心检索页）

布局：顶部导航+检索区，中间热点问题标签，下方导师列表

- 【顶部导航】（水平排列，居左）
       

    - 标题：高考志愿咨询师（加粗，居左）

    - 右侧：我的预约、个人中心（图标+文字）

- 【顶部检索区】（核心新增，水平排列，占满页面宽度）
        

    - 检索框：输入咨询关键词/热点问题（placeholder：输入关键词/热点问题，如“保研”“计算机就业”，居中）

    - 搜索按钮：搜索（加粗，居右）

- 【中间】热点问题标签（水平滚动，核心新增）
        

    - 标签内容：冲稳保怎么判断、热门专业薪资、保研流程、志愿表填写、转专业难度（每个标签可点击，点击后跳转至对应导师列表）

- 【中间】筛选栏（水平排列，可展开/收起）
        

    - 筛选选项：学校、专业、导师段位（青铜/白银/黄金/钻石/王者）、价格区间（50以下、50-100、100-200、200以上）

- 【下方】导师列表（网格布局，每行2-3个卡片，每个卡片包含以下元素）
        

    - 元素1：导师头像（圆形，居中）

    - 元素2：导师姓名（加粗，居中）

    - 元素3：标签（水平排列，居中）：身份标签（在校生/毕业3年内）+ 院校+专业 + 段位title（如“黄金咨询师”）

    - 元素4：关键词标签（水平排列，居中，如“保研、计算机、就业”）

    - 元素5：咨询单价（居中，加粗，如“100元/30分钟”）

    - 元素6：可预约最近时间（居中，灰色字体，如“最近可约：6月10日 19:00”）

    - 元素7：按钮（居中，加粗）：立即预约

## 页面2：导师详情页（核心转化页）

布局：顶部导师基础信息，中间资质+介绍，右侧固定预约区（PC端）；移动端垂直排列，预约区在底部固定

- 【顶部】导师基础信息（水平排列，居左）
        

    - 元素1：导师头像（圆形，较大）

    - 元素2：基础信息（垂直排列）
                

        - 姓名（加粗，大字体）

        - 段位title（如“黄金咨询师”，加粗，颜色突出）

        - 标签：身份标签 + 院校+专业

        - 咨询单价（加粗，如“100元/30分钟”）

        - 咨询人数（灰色字体，如“已服务120人”）

- 【中间左侧】资质与宣传材料
        

    - 标题1：咨询关键词（加粗），下方显示关键词标签（如“保研、计算机、就业”）

    - 标题2：自我介绍（加粗），下方显示导师自我介绍文本

    - 标题3：资质材料（加粗）
               

        - 图片展示（网格布局，点击可放大）：成绩、奖项、科研等图片

        - 视频展示（视频播放器）：导师宣传视频（无回放功能）

- 【右侧/底部固定】预约区（核心转化区）
        

    - 标题：预约咨询（加粗）

    - 字段1：选择日期（日历选择，仅显示导师可预约日期）

    - 字段2：选择时间段（下拉选择，显示该日期下导师可预约的时间段）

    - 字段3：预设问题（文本框，提示文字：请填写咨询的核心问题，帮助导师提前准备，必填）

    - 字段4：费用展示（加粗，如“本次咨询费用：100元（30分钟）”）

    - 按钮1：预付费用 · 确认预约（居中，加粗，颜色突出）

    - 按钮2：取消（居左，灰色）

    - 小字提示：预约规则：提前24小时取消全额退款，提前<24小时取消扣50%费用，未到场不退款

## 页面3：预约成功页

布局：垂直居中，从上到下依次排列，底部固定操作按钮

- 【顶部】提示图标+文字（居中）
        

    - 图标：预约成功图标（绿色对勾）

    - 文字：预约成功！（加粗，大字体）

- 【中间】预约详情（垂直排列，间距均匀，居中）
        

    - 字段1：导师姓名（如“张XX 黄金咨询师”）

    - 字段2：咨询时间（如“2026-06-10 19:00-19:30”）

    - 字段3：咨询费用（如“100元（30分钟）”）

    - 字段4：预设问题（如“咨询计算机专业保研流程”）

    - 字段5：倒计时（加粗，红色字体，如“距咨询开始还有23小时45分钟”，实时刷新）

- 【底部固定】操作按钮（水平排列，居中）
        

    - 按钮1：取消预约（居左，灰色；点击后弹出确认弹窗，提示违约规则）

    - 按钮2：进入会议（居右，加粗，颜色突出；预约时间前10分钟可点击，未到时间置灰）

## 页面4：视频会议页

布局：中间视频画面，底部操作栏，顶部标题

- 【顶部】标题：视频咨询（居中，加粗）+ 剩余时间（如“剩余28分钟”，实时刷新）

- 【中间】视频画面（占满页面大部分区域）
        

    - 左侧：导师视频画面

    - 右侧：用户视频画面（可隐藏）

- 【底部固定】操作栏（水平排列，居中）
        

    - 按钮1：开启/关闭麦克风（图标+文字）

    - 按钮2：开启/关闭摄像头（图标+文字）

    - 按钮3：结束会议（加粗，红色；点击后弹出确认弹窗，确认后结束会议）

- 【底部小字】提示：会议无回放，结束后将自动生成会议笔记

## 页面5：我的预约页

布局：顶部筛选栏，中间预约列表，每条列表右侧操作按钮

- 【顶部】筛选栏（水平排列）
       

    - 筛选选项：全部预约、待开始、已完成、已取消

- 【中间】预约列表（每条列表水平排列，包含以下字段）
        

    - 字段1：导师头像+姓名+段位title（如“张XX 黄金咨询师”）

    - 字段2：咨询时间（如“2026-06-10 19:00-19:30”）

    - 字段3：预约状态（待开始/已完成/已取消，不同状态不同颜色标注）

    - 字段4：咨询费用（如“100元”）

    - 【每条右侧】操作按钮（根据状态显示）

        - 待开始：取消预约 / 进入会议（到点可点击）

        - 已完成：查看会议笔记 / 提交服务反馈（仅提交满意/不满意+描述，不打分）

# 三、平台端（运营/管理端）核心页面

## 页面1：首页（数据概览页）

布局：顶部导航，中间数据卡片，下方核心操作入口

- 【顶部导航】（水平排列）：首页、咨询师管理、用户管理、预约管理、数据管理、关键词管理、协议管理、违规管理、培训系统

- 【中间】数据卡片（网格布局，每行3-4个）
        

    - 卡片1：咨询师总数（如“1200人”）

    - 卡片2：今日预约数（如“86单”）

    - 卡片3：累计咨询数（如“15600单”）

    - 卡片4：违规次数（如“12次”）

    - 卡片5：各段位咨询师分布（青铜/白银/黄金等数量）

- 【下方】核心操作入口（网格布局）
        

    - 入口1：咨询师审核（按钮+数字提示，如“待审核12人”）

    - 入口2：关键词审核（按钮+数字提示，如“待审核36条”）

    - 入口3：违规处理（按钮+数字提示，如“待处理5条”）

    - 入口4：评分裁决（按钮，进入裁决页面）

## 页面2：关键词库管理页（核心新增）

布局：顶部操作栏，中间关键词列表，底部分页

- 【顶部操作栏】（水平排列）
        

    - 标题：关键词库管理（加粗，居左）

    - 右侧：新增关键词（按钮）、批量删除（按钮）、搜索框（搜索关键词）

- 【中间】关键词列表（水平排列，表头+内容）
        

    - 表头：关键词名称、关键词类别（志愿填报/学业发展等）、使用次数、操作

    - 操作按钮：编辑、删除（每条关键词右侧）

- 【底部】分页（居中，显示总条数、当前页码、上下页按钮）

## 页面3：导师关键词审核页（核心新增）

布局：顶部筛选栏，中间审核列表，每条列表右侧审核按钮

- 【顶部筛选栏】（水平排列）
        

    - 筛选选项：待审核、审核通过、审核驳回

    - 搜索框：输入咨询师姓名/关键词检索

- 【中间】审核列表（每条列表包含以下字段）
        

    - 字段1：咨询师姓名、院校、专业

    - 字段2：自定义二级关键词（如“某院校计算机保研细节”）

    - 字段3：提交时间

    - 【每条右侧】审核按钮：通过 / 驳回（驳回需填写驳回原因）

## 页面4：导师评分裁决页（核心新增）

布局：顶部筛选栏，中间裁决列表，每条列表右侧裁决按钮

- 【顶部筛选栏】（水平排列）
        

    - 筛选选项：待裁决、已裁决

    - 搜索框：输入咨询师姓名检索

- 【中间】裁决列表（每条列表包含以下字段）
       

    - 字段1：咨询师姓名、当前段位、当前成长值
